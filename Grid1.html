COL TWO

  • Created a minHeight class in stylesheet, set the minimum height to 80vh (80% of device VIEW HEIGHT), and applied it to the container to make the row taller.
  • Set all of the col divs to h-100 (height 100% of container).
  • Set class "row-cols-1" to the row class so that there is only 1 column on mobile devices and larger. Then set "row-cols-md-2" immediately after so that it switches to 2 columns when screen size hits medium and larger.
  • Set padding to p-2 on both inner divs so text isn't right to edge.
